# BiomeMetric-Bukkit
A personal project designed for a community server.
A 1.21.4 plugin that overrides vanilla crop growth with customizable times per biome. Supports 1.21.4 Terralith biomes

## Biome Groups

This project organizes Minecraft biomes into different climate-based groups. Each group includes both vanilla and Terralith mod biomes.

### Tropical
Warm, humid biomes with dense vegetation:
- Vanilla: `JUNGLE`, `BAMBOO_JUNGLE`, `SPARSE_JUNGLE`, `MANGROVE_SWAMP`, `SWAMP`
- Terralith: `amethyst_rainforest`, `amethyst_canyon`, `jungle_mountains`, `rocky_jungle`, `tropical_jungle`, `orchid_swamp`, `alpha_islands`, `mirage_isles`, `lavender_forest`, `lavender_valley`, `moonlight_grove`, `moonlight_valley`, `sakura_grove`, `sakura_valley`, `blooming_plateau`, `blooming_valley`, `cloud_forest`

### Arid
Dry biomes with minimal rainfall:
- Vanilla: `DESERT`, `BADLANDS`, `ERODED_BADLANDS`, `WOODED_BADLANDS`, `SAVANNA`, `SAVANNA_PLATEAU`, `WINDSWEPT_SAVANNA`
- Terralith: `ancient_sands`, `arid_highlands`, `bryce_canyon`, `desert_canyon`, `desert_oasis`, `desert_spires`, `fractured_savanna`, `lush_desert`, `red_oasis`, `sandstone_valley`, `savanna_badlands`, `savanna_slopes`, `warped_mesa`, `white_mesa`, `ashen_savanna`, `brushland`, `hot_shrubland`, `shrubland`, `painted_mountains`, `stony_spires`, `temperate_highlands`, `volcanic_crater`, `volcanic_peaks`, `white_cliffs`, `basalt_cliffs`, `granite_cliffs`, `alpine_highlands`, `shield_clearing`, `valley_clearing`, `steppe`, `caldera`, `yellowstone`

### Temperate
Moderate climates with mixed forests and plains:
- Vanilla: `PLAINS`, `FOREST`, `FLOWER_FOREST`, `BIRCH_FOREST`, `DARK_FOREST`, `MEADOW`, `SUNFLOWER_PLAINS`, `OLD_GROWTH_BIRCH_FOREST`
- Terralith: `highlands`, `forested_highlands`, `shield`, `warm_river`, `skylands`, `skylands_autumn`, `skylands_spring`, `skylands_summer`, `lush_valley`, `mountain_steppe`, `yosemite_cliffs`, `gravel_beach`

### Chilly
Cold biomes with snow and coniferous forests:
- Vanilla: `TAIGA`, `OLD_GROWTH_PINE_TAIGA`, `OLD_GROWTH_SPRUCE_TAIGA`, `SNOWY_TAIGA`, `GROVE`, `SNOWY_PLAINS`
- Terralith: `alpha_islands_winter`, `birch_taiga`, `siberian_taiga`, `skylands_winter`, `ice_marsh`, `yosemite_lowlands`, `snowy_badlands`, `snowy_cherry_grove`, `snowy_maple_forest`, `snowy_shield`, `cold_shrubland`, `rocky_shrubland`, `siberian_grove`, `gravel_desert`, `windswept_spires`

### Alpine
High mountain biomes with snow and rocky terrain:
- Vanilla: `FROZEN_PEAKS`, `JAGGED_PEAKS`, `STONY_PEAKS`, `SNOWY_SLOPES`
- Terralith: `alpine_grove`, `frozen_cliffs`, `glacial_chasm`, `emerald_peaks`, `scarlet_mountains`, `rocky_mountains`, `haze_mountain`, `wintry_forest`, `wintry_lowlands`
